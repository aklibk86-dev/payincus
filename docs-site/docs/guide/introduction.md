# 项目介绍

PayIncus 是基于 Incus 的 NAT VPS 销售、交付与管理平台，面向托管商、节点运营者和需要自助交付面板的团队。

系统包含三类主要角色：

- 访客和用户：浏览套餐、注册登录、购买实例、使用终端、提交工单、管理钱包和通知。
- 管理员：管理用户、套餐、节点、镜像、账务、系统配置、通知、日志和在线更新。
- 宿主机 Agent：上报宿主机资源、实例状态、流量数据，并参与交付链路。

## 当前重点

- 非 Docker 生产部署。
- 用户端和管理端分离。
- 同源 `/api` 和 `/api/ws` 反代。
- 支付回调、余额和资源交付安全。
- Agent 安装、心跳和资源上报。
- 后台在线更新和原子 OTA。
